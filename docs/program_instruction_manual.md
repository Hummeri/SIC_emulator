## 파일 읽기
파일을 읽어들여서 스페이스가 나오면 그냥 무시하고, 텍스트가 나오면 다음 스페이스가 나올때까지 문자를 읽어 들인 다음, 미리 저장된 명령어 중에 일치하는 것이 있는지 찾아본다.
일치하지 않으면 에러 메세지를 보여주고 에셈블링을 아예 하지 않는다.
